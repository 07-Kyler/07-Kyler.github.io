# 题目：[P1577 切绳子](https://www.luogu.com.cn/problem/P1577)

**标签：** #二分答案 #二分模板题
## 题意描述

有 N 条绳子，它们的长度分别为 $L_i$ ​。如果从它们中切割出 K 条长度相同的绳子，这 K 条绳子每条最长能有多长？答案保留到小数点后 2 位(直接舍掉 2 位后的小数)。
 
切割后每条绳子的最大长度。答案与标准答案误差不超过 0.01 或者相对误差不超过 1% 即可通过。

## 解题思路

也是一道二分答案 二分逼近最长的绳子长度  用这个长度整除每一条绳子得到每一条绳子可分的条数 最后返回 $cnt\geq k$ 的判断值 再进行二分

题目提示了是浮点二分所以维护边界的时候不要+-1

---

${{\Huge Code}}$
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,k;
vector<double> lens;

bool check(double len);

int main (){
    cin>>n>>k;
    lens.resize(n);
    double mx = 0.0;
    for(int i = 0;i<n;++i){
        cin>>lens[i];
        mx = max(mx,lens[i]);
    }
    
    int cishu = 100;
    double ans = 0.0;
    double l = 0.0,r = mx+1;
    while(cishu--){//这里因为是浮点二分所以维护边界的时候直接等于中值
        double mid = (l+r)/2;
        if(check(mid)) l = mid;
        else r = mid;
        ans = l;

    }
    
    printf("%.2f",ans);
    return 0;


}
bool check(double len){
    int sum = 0;
    for(int i = 0;i<n;++i){
         sum+=(int)(lens[i]/len);//check():传入二分出来的长度参数 循环每一条绳子的长度
                                 //每一条绳子整除长度之和与k作比较如果大于k说明这个长度还能再大
    }
    return sum>=k;

}
```
